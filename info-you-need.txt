# Info you need

## Voraussetzungen
- Node.js (>= 20.x) installieren und im Pfad verfügbar machen.
- npm (wird mit Node ausgeliefert).

## Projekt vorbereiten
```bash
npm install
```

## Tests ausführen (führt automatisch `npm run lint` via `pretest` aus)
```bash
npm test
```

## Code-Qualität prüfen
```bash
npm run lint
```

## Alles prüfen (Alias für `npm test`)
```bash
npm run verify
```

## Empfohlene Zusatztools (optional, für spätere Ausbaustufen)
```bash
npm install --save-dev vite typescript eslint prettier
```
